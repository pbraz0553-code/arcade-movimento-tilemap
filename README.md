# Movimento Tilemap – MakeCode Arcade

Extensão para controlar movimento de personagens usando o controle
e tilemap no MakeCode Arcade.

## Blocos disponíveis
- Ativar movimento do jogador
- Parar movimento do jogador
- Ativar colisão com tilemap

## Como usar
1. Abra um projeto no MakeCode Arcade
2. Vá em Extensões
3. Cole a URL deste repositório
4. Use os blocos
